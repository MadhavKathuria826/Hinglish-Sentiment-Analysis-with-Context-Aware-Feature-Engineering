# 💬 Hinglish Sentiment Analysis (Research in Progress)

This project explores sentiment analysis of **Hinglish (code-mixed Hindi-English text)** using classical machine learning enhanced with **context-aware feature engineering**.

---

## 📌 Overview

Hinglish text presents unique challenges for NLP systems due to:

* spelling variations (e.g., *acha, accha, achha*)
* informal grammar
* code-mixing within sentences
* slang and non-standard vocabulary

This work investigates how traditional ML models can be improved through **domain-specific preprocessing and contextual features**.

---

## 🧠 Approach

* Hinglish-specific normalization and semantic mapping
* Handling spelling variations and slang
* Negation-aware feature engineering
* TF-IDF vectorization with n-grams
* Evaluation of:

  * Logistic Regression
  * Naive Bayes
  * Support Vector Machines

---

## 📊 Dataset

* ~25,000 Hinglish text samples
* Noisy, real-world social media data
* Adapted for sentiment classification

---

## ⚠️ Current Status

> This project is currently in an **early research phase**.

* Methodology under development
* Dataset preprocessing evolving
* Full implementation and experiments pending

---

## 🎯 Goal

To evaluate whether **context-aware feature engineering** can significantly improve classical ML performance on code-mixed language tasks without relying on deep learning.
